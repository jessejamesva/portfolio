Monster in a Box

This is my version of a Monster-in-a-Box Holloween prop. It is driven by an Arduino microcontroller and powered by 12v and 5v power supplies. It uses a windshield wiper motor to turn a homemade 'sprocket' that lifts the lid about 1/2" before loudly dropping it back down. The chains rattle and provide additional sound/scare factor.

This is a continous project with different interations based on how it is going to be used. It can be operated with a proximity sensor or a double-throw switch. It also now has an audio track that plays a warewolf growl on a constant loop. The audio is through an external bluetooth speaker but next year I hope to add an attached system and maybe a phone app controller.

Arduino, C++, Basic Electricity

Color Organ

A Color Organ is a reactive light instrument that filters different frequencies from a sound wave. These are my attempts. The first (not pictured) was a smaller version and both were from schematics I found online. The test was done on a breadboard and then a permanent one was made on a protoyping board. This project is limited from a coding aspect but the learning value was massive.

Schematic Reading, Basic Electronics, Linear Devices

Arduino Alarm Clock

This was a final project build for a college microprocessor class. It is powered by an Arduino microcontroller and written in C++. Besides the contorller, the other modules are an 16x2 LCD, a real-time clock, and a temperature/humidty sensor. There is stopwatch function that can be run in conjunction with other features. This was an excellent project that allowed me to practice circuit and program design. It also taught me the importance of breaking large projects into smaller, more managable modules.

Arduino, C++, Electronics, Design
